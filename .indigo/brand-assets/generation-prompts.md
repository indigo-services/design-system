# Generation Prompts

## `logo-wordmark-master`
Create a flat SVG wordmark for `INDIGO`. Use a simple circular phone mark on the left and a clean horizontal wordmark on the right. Tone: enterprise service brand, calm, precise, small-size legible. Palette: `#1F2D8A`, `#24379F`, `#DCE7FF`, `#F7F8FD`, `#18253A`. Transparent background. No gradients, no bevels, no 3D, no Strapi-style purple.

## `logo-wordmark-light`
Create a light-mode SVG brand lockup for `INDIGO` intended for the Storybook sidebar header. Use the circular phone mark on the left, a dark Indigo wordmark on the right, and keep the output simple and flat. Transparent background.

## `logo-wordmark-dark`
Create a dark-mode SVG brand lockup for `INDIGO` intended for the Storybook sidebar header. Use a white circular phone badge with an Indigo handset and a white wordmark on the right. Transparent background. Flat vector only.

## `favicon-phone-mark`
Create a simple SVG favicon for Indigo using only the circular phone mark. Requirements: transparent background outside the mark, crisp at 16x16 and 32x32, flat vector only, slightly bluer Indigo than the current Strapi purple.

## `brand-symbol`
Create a compact SVG symbol mark for Indigo that can replace the current square Strapi symbol export. Requirements: square composition, readable at 16x16 and 32x32, flat vector only, transparent background, no gradients, no thin decorative lines, no purple branding. Use Indigo palette values and keep the silhouette distinctive.

## `welcome-cover`
Create a PNG hero cover image for the Storybook welcome page for `Indigo Services Design System`. Show a refined enterprise design-system atmosphere: interface fragments, layout rhythm, brand panels, and Indigo accents. Keep it editorial and product-oriented, not marketing-glossy. Avoid stock people, avoid purple, avoid screenshots of unrelated products. Output should work as a wide docs cover image.

## `legacy-brand-image`
Create a small replacement PNG for a legacy Strapi-branded docs image. The replacement should be a minimal Indigo brand graphic or badge-style product illustration that fits docs content without drawing too much attention. Keep it clean, flat, and compatible with Storybook docs surfaces.
